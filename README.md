# Parametric Residential Framing Tool — Schema & Design Docs

A constrained BIM engine for parametric residential framing. This repository currently holds the **design schema** for the engine — domain models, the architecture, and the analysis that consolidated them. No application code yet.

All documents live under [`docs/`](docs/). Start there:

- **Current schema** → [`docs/schema/unified-schema.html`](docs/schema/unified-schema.html) — the single, cohesive, future-proof domain model (open in a browser). Machine contract: [`docs/schema/unified-model.json`](docs/schema/unified-model.json).
- **Index & status of every document** → [`docs/README.md`](docs/README.md).
