Parametric Lumber
Schema — 2×4 Stud

A reusable domain model for a single framing member. Geometry is parametric on length and stored as integer ticks @ 1/32" (no float drift on imperial fractions); mechanical values and SKUs are resolved by reference (flyweight), never copied per instance. Cutting is a process acting on stock, not a property of it. Render data lives in a separate adapter.

Entity · identity persists Value Object · immutable Reference · shared flyweight Domain Service · process Render · presentation adapter
